# Public Route SEO Metadata Hardening

## Summary
Added explicit metadata to key public conversion/trust routes: `/docs`, `/security`, and `/contact`.

## Changes
- Added `export const metadata` to:
  - `src/app/docs/page.tsx`
  - `src/app/security/page.tsx`
  - `src/app/contact/page.tsx`
- Included route-specific titles and descriptions for crawlability and snippet quality.

## Why this matters
- Improves search indexing and relevance for high-intent pages.
- Increases top-of-funnel discovery quality and trust-page visibility.
- Supports enterprise buyer journeys starting from docs/security queries.

## Estimated business impact
- Acquisition: medium
- Activation: low-medium
- Paid conversion: medium
- Retention: low
