# MISSION.md — DocuMint AI Deployment Mission

## North Star

Build the most reliable AI coding + documentation IDE for teams, with enterprise security controls and measurable revenue impact.

## Deployment Standard

A feature is not complete until it is:
1. Implemented with strict typing and validation
2. Security-reviewed for auth/authz/input handling
3. Build-verified and regression-checked
4. Observable (errors are actionable and structured)
5. Documented with user/business impact

## Product Priorities (Current)

1. Convert more users to paid subscriptions
2. Improve end-to-end IDE reliability (editor, terminal, preview, file ops)
3. Harden IDE-adjacent API routes and integrations
4. Strengthen WebContainer lifecycle and fault recovery
5. Expand Mermaid architecture workflows safely

## KPI Targets

- Time-to-first-successful-project creation
- Trial/free to paid conversion rate
- Weekly active IDE users
- API route hardening coverage
- Production error rate and MTTR
